# CodeInsight AI

An AI-powered code review platform that combines static code analysis with Generative AI to identify bugs, security issues, performance bottlenecks, and provide intelligent refactoring suggestions.

## Tech Stack

### Frontend

- React
- TypeScript
- Tailwind CSS
- Monaco Editor

### Backend

- Node.js
- Express
- Prisma ORM
- PostgreSQL
- JWT Authentication

### AI

- Google Gemini API

### Static Analysis

- ESLint
- Pylint (Coming Soon)

## Features

- Authentication
- Code Snippet Review
- File Upload Review
- AI Code Analysis
- Static Analysis
- Review History
- Analytics Dashboard
- Documentation Generator
